Databases

In the CS-210 Programming Languages course, I created a C++ command line application designed to analyze text records of items purchased and provide the user with valuable insights. The three main features are searching for specific item frequency, generating a list of all items with their frequency, and creating a histogram of all the items.

Originally this application just stored and read data from text files; my enhancement improves this application by using Google Firebase to store user data for user authentication along with their product data. This allows users to utilize this application from wherever they have internet access as their data will always be securely stored in the cloud.
